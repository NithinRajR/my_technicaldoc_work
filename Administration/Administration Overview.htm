Administration Overview

As an Administrator, you can create and maintain all static data.

You can also create and maintain users.

In this module, the Administrator can configure all static data that applies to both the modules: ABL and Factoring.

Navigation: Administration → Administration Overview

The static data common to the ABL and Factoring modules are as follows:

  • Lender Details

    • Basic Details

    • Bank Details

  • General Codes

    • Sales Regions

    • NAICS Codes

    • Comment Types

    • System Parameters

    • Adjustment Reasons

    • Interest Calendar

    • Content

    • Report Templates

  • User Maintenance

    • User List

    • Roles & Permissions

    • Client Assignment

    • User Defined Field

  • Admin Logs

    • Audit Logs

    • Automated Task Logs

    • Notification Logs

  • Notification Center

    • Notification Settings

    • Templates
