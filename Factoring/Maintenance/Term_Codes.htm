Term Codes

The Term Code is used to calculate the due date of invoices in the Factoring application.

This page allows users to define term codes, which are then used throughout the application for determining invoice due dates.

Fields & Descriptions:

Fields

Descriptions
Status 

A dropdown filter that narrows the page data by All, Active, or Inactive.

  • Active term codes are currently in use and appear in the Term Codes dropdown throughout the application.
  • Inactive term codes have been disabled by the user and are hidden from all Term Codes dropdowns across the application.
Global Search

Searches across all term codes and lists the closest matching records on screen.

Name

The name of the term code as it will appear in the Term Codes dropdown throughout the application.

Searchable via the grid search option.
Days

The number of days used to calculate the due date from the invoice date.

Searchable via the grid search option. 

Example:

  • Invoice Date: 15-01-2025
  • Term Codes: 30 days
  • Due Date: 14-02-2025 (15-01-2025 + 30 Days)
Status (Grid Column)

 Displays the current status of each term code. Searchable via the grid search option.
