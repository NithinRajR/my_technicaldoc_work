Invoice Status

The Invoice Status page allows you to create and manage the invoice statuses used across the Factoring module.

Statuses drive invoice processing workflows, and any status you define here is immediately available throughout the application.

    To add a new Invoice Status, follow the steps below:

    1. Click the  button to summon the inline editor.

    2. Enter Name and Description.

    3. Click the  button to save the record.

    4. After a successful save, the message "Invoice Status was successfully created" is displayed on the screen.

    Note:

    • Code and Name must be unique.
    • If a duplicate value is entered in either field, the system displays an error and prevents the record from being saved.

  • Edit an Invoice StatusEdit an Invoice Status

    To edit an existing Invoice Status, follow the steps below:

    1. Double-click any row in the grid to enable edit mode for that record.

    2. Make your changes to the fields as required.

    3. Click the  button to save the record.

    Refer to the screenshot:

    4. After a successful save, the message "Invoice Status was successfully updated" is displayed on the screen.

    Note:

    • Only the Name and Description fields can be modified.
    • The Code field is locked after the record is created.

Fields & Descriptions:

Fields

Descriptions
Code

A unique identifier for the invoice status.

The grid includes a search filter to quickly locate a specific code. This field cannot be edited after the record is saved.
Name

The display name of the invoice status.

Must be unique across all records. The grid includes a search filter to locate a status by name.

Description

A free-text description of the invoice status.

This field is optional but recommended for clarity in processing workflows.
