Data Upload

The Data Upload section displays all files with invoice records that users uploaded through the Factoring File Uploads screen.

Users upload invoice files through the Factoring File Uploads screen, using mappings created in the Mapping List View module.

Here is a quick look at the application page: 

Fields & Descriptions:

Fields

Descriptions

File Name

Displays the uploaded file name as a hyperlink.

Click it to open the New Invoices page and view the file’s invoice details.
Expected Value

Shows the total invoice amount entered during upload.
Actual Value

Shows the total invoice amount system-calculated from the uploaded file.

Variance

Displays the difference between the Expected Value and the Actual Value.

Schedule Number

Displays the schedule number as a hyperlink. Clicking it opens an HTML report of the uploaded invoice data.

Refer to the screenshot:
User

Shows the display name of the user who uploaded the file.

Start Time

Displays the file upload start timestamp in MM/DD/YYYY HH:MM:SS format.

End Time 

Displays the file upload completion timestamp in MM/DD/YYYY HH:MM:SS format.

Created At

Shows the timestamp for when the file record was created on the New Invoices page.

New Account

Displays the total number of newly created account debtors. These accounts appear in the Account Debtors page.

Original Records

 Shows the count of original invoice records as a hyperlink. Clicking it navigates to the New Invoices page to view the details.

Duplicate Records

Shows the count of duplicate invoice records as a hyperlink. Clicking it opens the Duplicate Invoices page.

Total Records

Displays the total number of records in the upload (Original + Duplicate).

Actions 

The  icon deletes the uploaded invoice file.

  • Clicking Undo shows a confirmation message: “Are you sure?”

  • When confirmed, the system removes the upload entry from both the Data Upload and New Invoices pages.
  • A notification related to the undo action appears under the bell icon.

Note:

  • Newly created account debtors are not deleted. Only the invoice records (original and duplicate) are removed.
  • Ask your administrator if you don't see an  icon against the Actions field, as this is purely a role based access.
