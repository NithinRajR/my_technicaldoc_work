Factoring File Uploads

The Factoring File Upload feature allows users to upload a number of Account Debtors and Invoices in a single operation.

This simplifies the factoring workflow by enabling bulk data imports rather than manual entry.

Before uploading, each file format must be mapped in the system through:

Client Maintenance → Client Data Mappings → Mapping List View.

Once a mapping is created, the corresponding file type becomes available for upload on the Factoring File Upload page.

After the upload is completed, users can review the results in:

  • Data Upload – to verify the upload status and processed records
  • New Invoices – to view all newly uploaded invoices
  • Account Debtor page – to review uploaded account debtor details

Any file format other than CSV, XLS, XLSX, or XLSM is automatically forwarded to CYNC Support for mapping through MonarchMonarch.

CYNC doesn’t support PDF, RPT, TXT, or HTML files for direct upload.

To handle these formats, CYNC uses Monarch, a third-party integration that converts such files into CSV so they can be uploaded into the system.

This conversion service is enabled based on a Lender - CYNC agreement that includes predefined billing terms.

  • Learn how the duplicate files are managedLearn how the duplicate files are managed

    CYNC generates a new Account Number for each repeated Account Name by appending “-1,” “-2,” “-3,” etc., while enforcing the 20-character limit.

    Here is an example:

    Upload 1
    CYNC truncates CANADRIAL INDUSTRIES PRIVATE LIMITED to CANADRIAL INDUSTRIES (due to the 20-character limit) and creates the initial account.

    Upload 2
    You upload the same file again with new invoice numbers. CYNC detects a duplicate and creates CANADRIAL INDUSTRIES-1

    Upload 3
    You upload the file again. CYNC creates CANADRIAL INDUSTRIES-2 as the next available version.

Fields

Descriptions

Data File Type

 Specifies the file type you selected during the mapping process.
Mapping Name

Specifies the name of the mapped file.Specifies the name of the mapped file. 
Format

Specifies the format of the uploaded file. Supported formats include:

  • CSV
  • XLS
  • XLSX
  • XLSM

Note: Any other file format is automatically forwarded to Cync Support for mapping in Monarch.

Expected Value

Allows you to manually enter the total dollar amount or expected value of the invoices.

Note: If you leave the Expected Value field blank or enter a value that does not match the total invoice amount in the file, the file will still process successfully. However, you will be notified of the same.
