Cash Reserve

The Cash Reserve page displays all transactions that impact the Reserve Balance.

This includes charge and chargeback transactions, as well as interest entries generated based on the configuration defined on the Interest Setup page.

Fields & Descriptions:

Fields

Descriptions
Account Number

Displays the account number associated with the invoice.
Account Name

Displays the account name associated with the invoice.

The value appears as a hyperlink and allows users to navigate to

  • Invoice Details
  • Account Parameters
  • Comments
  • Invoice Documents.
Invoice Number

Displays the account number of the debtor associated with the invoice.
PO Number

Displays the purchase order number associated with the invoice.

Invoice Date

Displays the invoice date in MM/DD/YYYY format.
Due Date

Displays the invoice due date in MM/DD/YYYY format.

The due date is calculated based on the invoice date and the payment terms configured for the invoice.

Funding Batch

Displays the funding batch number generated on the Verified page.

Fund Date

Displays the date on which the invoice batch was funded, in MM/DD/YYYY format.

The fund date is selected in the Fund Batch pop-up on the Verified page.

Fee Name

Displays the fee name associated with the client.

Reserve Date

Displays the date the invoice entry was recorded on the Cash Reserve page, in MM/DD/YYYY format.

Transaction Type

Displays the transaction type recorded on the Cash Reserve page. Supported values include:

  • PMT (Payment)
  • PP (Partial Payment)
  • SP (Short Payment)
  • OP (Over Payment)
  • NF and NF-UAC
  • CB (Chargeback – partial or complete)

Transaction Date

Displays the batch process date of the invoice, in MM/DD/YYYY format.

Reference No

Displays the payment reference number associated with the transaction.

Invoice Amount

Displays the total invoice amount.

Payment Amount

Displays the payment amount applied to the invoice.

Advance Amount

Displays the advance amount funded for the invoice.

The advance amount is calculated based on the percentage configured in the Factoring Fee Setup page.
Accrued Fee

Displays the accrued fee amount for the invoice, if tier-based fee setup is configured.
Fee Days

Displays the fee days calculated based on the setup in the Factoring Basic Parameters page.

Fee days are calculated from either the Invoice Date or the Fund Date, based on the client configuration.
