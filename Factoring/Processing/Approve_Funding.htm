Approve Funding

The Approve Funding screen displays all invoices associated with a selected funding batch. From this screen, you can review invoice details and perform the following actions:

  • Approve – Approves the entire funding batch.
  • Reject – Rejects the funding batch.
  • Print – Generates and prints the Funding Schedule Report, which includes the complete list of invoices in the batch.

When a funding batch is sent for approval using the  button in the Fund Batch pop-up of the Verified page, an email notification containing a link is sent to the designated approver.

When the approver clicks the link, they are redirected to the Approve Funding page, where all invoices associated with the selected funding batch are displayed for review and approval.

Approval Workflow

  1. In the Fund Batch pop-up, click the  button from the Verified page.
  2. The system sends an emailemail to the assigned approver with a secure link.

    4. Enter the name of the authorized approver.

Fields & Descriptions:

Fields

Descriptions

Invoice(s)

Lists the invoice number for each invoice included in the funding batch.
Account Number

Displays the debtor’s account number associated with each invoice.
Account Name

Displays the debtor’s account name for each invoice.

Invoice Date

Displays the invoice date in MM/DD/YYYY format.

Due Date

Displays the due date of the invoice in MM/DD/YYYY format.

Invoice Amount

 Displays the original amount of the invoice.
Advance %

Displays the advance percentage applied to the invoice.
Advance Amount

Displays the amount advanced against the invoice.

Base Fee

Displays the base fee charged for the invoice.
Spread Fee

Displays the spread fee charged for the invoice.
Earned Upfront Fee

Displays the upfront fee earned for the invoice. Negative values appear in brackets.
Net Purchase Price

Displays the net purchase price calculated as:

Advance Amount – Earned Upfront Fee.
Disbursement Method

Displays the method used to disburse funds to the client.

Bank Name

 Displays the client’s bank name for disbursement.

Bank Account Number

 Displays the client’s bank account number.

Routing Number

Displays the bank routing number.

Total Invoice Amount

Displays the total of all invoice amounts in the batch.
Total Advance Amount

Displays the total advance amount for the batch.

Reserve Amount

Displays the total reserve amount for the batch.

Total Earned Upfront Fee

 Displays the total earned upfront fees for the batch. Negative values appear in brackets.
Purchase Price %

Displays the purchase price percentage.
Base Rate %

Displays the base rate percentage.

Spread Rate %

Displays the spread rate percentage.

Total Fee %

Displays the total fee percentage.

Reserve Release

Displays the reserve rebate amount.

Total Disbursement

Displays the total amount to be disbursed to the client, calculated as:

Total Advance Amount – Total Earned Upfront Fee.
