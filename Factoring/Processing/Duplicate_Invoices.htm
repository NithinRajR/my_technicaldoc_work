Duplicate Invoices

An invoice is marked as Duplicate when its invoice number matches an existing invoice in the system or another invoice within the same file being processed through the Factoring File Uploads page and shown on the Data Upload page.

Duplicate invoices appear on the Duplicate Invoices page only when they come from the Factoring File Uploads.

Note: The Manual Entry process does not allow duplicate invoices to be submitted.

Each duplicate invoice listed on the duplicate invoices page includes Accept, Edit, and Delete options, available through radio buttons.

Fields & Descriptions:

Fields

Descriptions

Invoice Number

The system displays the invoice number for each record processed through Manual Entry, Factoring File Uploads, or QuickBooks imports.
Account Number

This column displays the debtor’s account number for each invoice listed on the New Invoices page.
Account Name

This column displays the debtor’s account name for each invoice.

Invoice Date

Displays the invoice date in MM/DD/YYYY format.

PO Number

Displays the purchase order number for the invoice.

Amount

Displays the invoice amount as a numeric value with two decimal places.
This column displays the computed due date in MM/DD/YYYY format.

The system calculates the due date using the invoice date and the number of days defined in the selected payment term.

Example:

  • Invoice Date: 01/05/2021

  • Payment Terms: Net 30 (30 days)

  • Due Date = 01/05/2021 + 30 days = 01/31/2021
